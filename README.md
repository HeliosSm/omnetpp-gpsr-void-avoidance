# omnetpp-gpsr-void-avoidance
Implementación de una mejora al protocolo GPSR para redes WSN/MANET en OMNeT++ (INET Framework). Sustituye el "Perimeter Mode" por un mecanismo híbrido de Backpressure y Recuperación Lateral (Fallback) para resolver el problema de mínimos locales (vacíos) sin planarización de grafos.
